# TARKIR: DRAGONSTORM · TDM

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **ARENA**.

TARKIR: DRAGONSTORM (TDM) — an ARENA Magic: The Gathering sphere: Magic's 2025 return to Tarkir, the five dragon clans under a brewing storm, curated by basic worth. Pulls all ~277 cards LIVE from Scryfall (card data + the non-foil USD it mirrors from TCGplayer) and buckets them into worth tiers — 🩸 chase ($5+) · staples ($1–5) · role-players ($0.25–1) · bulk (&lt;$0.25) — with a set-value summary (sum / top card / chase count), search, rarity filter, foil-worth toggle, and a Scryfall thumbnail per card. The curation is the tiering + the read applied to REAL numbers (verified live: 277 cards, ~$280 non-foil set value, top Ugin Eye of the Storms ~$39.84 + Elspeth Storm Slayer ~$38.70, 9 chase). Dragonstorm palette (dragonfire amber + storm blue). ⚑ the new return set (tdm, Apr 2025), distinct from classic Khans (ktk) / Dragons of Tarkir (dtk). Honest: prices are a live snapshot and fluctuate daily, an unlisted card shows —; card names &amp; prices are facts, images hot-linked from Scryfall's CDN (sanctioned non-commercial use); an unaffiliated fan price-reference — Magic is © Wizards of the Coast.
